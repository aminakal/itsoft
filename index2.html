Айнымалылар

Айнымалы - деректерді сақтау үшін пайдаланылатын атау. Python тілінде айнымалыны анықтау өте оңай.

Мысал


# Айнымалыны анықтау
x = 5
y = "Сәлем"
print(x)
print(y)

Функциялар

Функциялар - кодты қайта пайдалануға мүмкіндік беретін блоктар. Python тілінде функцияларды def кілт сөзімен анықтайды.

Мысал


# Функцияны анықтау
def сәлемдесу():
    print("Сәлем, әлем!")

# Функцияны шақыру
сәлемдесу()

Шарттар

Шартты операторлар белгілі бір шарттың орындалуына қарай кодтың орындалуын басқарады. Python тілінде шартты операторлар if, elif және else кілт сөздерімен анықталады.

Мысал


# Шартты оператор
a = 10
b = 20

if a < b:
    print("a b-дан кіші")
elif a == b:
    print("a мен b тең")
else:
    print("a b-дан үлкен")

Циклдар

Циклдар белгілі бір шарттар орындалғанға дейін кодты қайталап орындауға мүмкіндік береді. Python тілінде циклдарды for және while кілт сөздерімен анықтайды.

Мысал


# for циклі
fruits = ["алма", "банан", "шабдалы"]
for fruit in fruits:
    print(fruit)

# while циклі
i = 1
while i < 6:
    print(i)
    i += 1

Кітапханалар

Python-да көптеген кітапханалар жасалған, оларды қолдану арқылы функцияларға, деректердің өңделуіне және басқа мүмкіндіктерге ие болуға болады.

Алдыңғыда осы жағдайларды қолдану үшін сіз алдыңғыларды орнату қажет.

Кітапханаларды орнату үшін төмендегі командаларды пайдаланыңыз:

  • pip install library_name
