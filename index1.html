HTML негіздері

HTML (HyperText Markup Language) – веб-беттердің құрылымын жасау үшін қолданылатын белгілеу тілі.

Мысал


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Үйренуге арналған бет</title>
</head>
<body>
    <h1>HTML және CSS әлеміне қош келдіңіз</h1>
    <p>Сіздің бірінші HTML-құжатыңыз.</p>
</body>
</html>

Негізгі тегтер HTML

  • Тақырыптар: <h1> - <h6>
  • Параграф: <p>
  • Сілтеме: <a href="https://example.com">Текст ссылки</a>
  • Сурет: <img src="image.jpg" alt="Описание изображения">
  • Тізімдер: реттелген <ol>, реттелмеген <ul>, элементы списка <li>

Семантикалық тегтер HTML5

HTML5 веб-беттердің құрылымы мен қолжетімділігін жақсарту үшін жаңа семантикалық тегтерді енгізді.

  • <header> — бет немесе бөлім тақырыбы.
  • <nav> — сайтты шарлау.
  • <main> — құжаттың негізгі мазмұны.
  • <article> — тәуелсіз мазмұн бөлігі.
  • <section> — құжаттың бөлімі.
  • <aside> — бүйірлік тақта немесе қолдаушы мазмұн.
  • <footer> — Беттің немесе бөлімнің төменгі деректемесі.

CSS негіздері

CSS (Cascading Style Sheets) иHTML-элементтерді әрлеу үшін қолданылады.

CSS пен HTML-ді қосу

  • Ішкі файл (внутри тега <style> в разделе <head>).
  • Сыртқы файл (через тег <link> в разделе <head>).
  • Кіріітірілген стиль (через атрибут style в HTML-элементе).

Пример CSS-кода


body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

header {
    background: #333;
    color: #fff;
    padding: 1rem 0;
    text-align: center;
}

main {
    padding: 1rem 2rem;
}
