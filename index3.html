Айнымалылар

Айнымалы - деректерді сақтау үшін пайдаланылатын атау. C# тілінде айнымалыны анықтау өте оңай.

Мысал


// Айнымалыны анықтау
int x = 5;
string y = "Сәлем";
Console.WriteLine(x);
Console.WriteLine(y);

Методтар

Методтар - функциялардың C# тіліндең айнымалысы. Олар кодты бөліп жатқан блоктарды қайта пайдалануға мүмкіндік береді.

Мысал


// Методты анықтау
void Сәлемдесу()
{
    Console.WriteLine("Сәлем, әлем!");
}

// Методты шақыру
Сәлемдесу();

Шарттар

Шартты операторлар белгілі бір шарттың орындалуына қарай кодтың орындалуын басқарады. C# тілінде шартты операторлар if, else if және else кілт сөздерімен анықталады.

Мысал


// Шартты оператор
int a = 10;
int b = 20;

if (a < b)
{
    Console.WriteLine("a b-дан кіші");
}
else if (a == b)
{
    Console.WriteLine("a мен b тең");
}
else
{
    Console.WriteLine("a b-дан үлкен");
}

Циклдар

Циклдар белгілі бір шарттар орындалғанға дейін кодты қайталап орындауға мүмкіндік береді. C# тілінде циклдарды for, while және do-while кілт сөздерімен анықтайды.

Мысал


// for циклі
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}

// while циклі
int j = 0;
while (j < 5)
{
    Console.WriteLine(j);
    j++;
}
