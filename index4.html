Айнымалылар

Айнымалы - деректерді сақтау үшін пайдаланылатын атау. JavaScript тілінде айнымалыны анықтау өте оңай.

Мысал


// Айнымалыны анықтау
var x = 5;
var y = "Сәлем";
document.write(x);
document.write(y);

Функциялар

Функциялар - кодты қайта пайдалануға мүмкіндік беретін блоктар. JavaScript тілінде функцияларды function кілт сөзімен анықтайды.

Мысал


// Функцияны анықтау
function сәлемдесу() {
    document.write("Сәлем, әлем!");
}

// Функцияны шақыру
сәлемдесу();

Шарттар

Шартты операторлар белгілі бір шарттың орындалуына қарай кодтың орындалуын басқарады. JavaScript тілінде шартты операторлар if, else if және else кілт сөздерімен анықталады.

Мысал


// Шартты оператор
var a = 10;
var b = 20;

if (a < b) {
    document.write("a б-дан кіші");
} else if (a == b) {
    document.write("a мен b тең");
} else {
    document.write("a б-дан үлкен");
}

Циклдар

Циклдар белгілі бір шарттар орындалғанға дейін кодты қайталап орындауға мүмкіндік береді. JavaScript тілінде циклдарды for, while және do-while кілт сөздерімен анықтайды.

Мысал


// for циклі
for (var i = 0; i < 5; i++) {
    document.write(i);
}

// while циклі
var j = 0;
while (j < 5) {
    document.write(j);
    j++;
}
